# DasIoT React

# Installation for development
1. Install packages
```sh
yarn
```
2. Install peer dependencies
```sh
yarn install-peers -f
```
3. Open Storybook
```sh
yarn storybook
```

# Usage
1. Install package
```sh
git+ssh://
```
2. Import Component
```tsx
import Button from 'dasiot-react';

const App = () => {
  return (
    <Button />;
  );
}
```
